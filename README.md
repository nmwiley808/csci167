---

## Course Overview

CSCI 167 introduces fundamental concepts, techniques, and algorithms in machine learning and deep learning. The primary goals of the course include:

- Understanding the mathematical and conceptual foundations of deep learning
- Learning how neural networks are trained using backpropagation
- Identifying and addressing common training challenges (e.g., vanishing gradients)
- Implementing modern deep learning architectures
- Gaining practical experience through experimentation and projects

---

## Topics Covered

- Machine Learning fundamentals
- Logistic regression
- Neural networks and backpropagation
- Vanishing / diminishing gradients
- Optimization techniques and normalization
- Batch normalization
- Residual networks (ResNets)
- Convolutional Neural Networks (CNNs)
- Recurrent Neural Networks (RNNs)
- Experimental evaluation of deep learning models
